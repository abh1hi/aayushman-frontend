### 3.1 Metadata Rules
Every page MUST include:
- One unique `<title>` (50–60 characters)
- One unique meta description (140–160 characters)
- Clean, readable URL slug (kebab-case)
- Canonical URL
- Open Graph & Twitter meta tags

No duplicated metadata across pages.

---

### 3.2 Heading Structure Rules
- Exactly ONE `<h1>` per page
- `<h1>` must include the primary keyword naturally
- Use `<h2>` for major sections
- Use `<h3>`–`<h6>` only for logical nesting
- Never skip heading levels

---

### 3.3 Content Quality Rules
- Minimum content length:
  - Landing pages: 700+ words
  - Blog / resource pages: 1,200+ words
  - Core service pages: 900+ words
- Content must:
  - Answer user questions clearly
  - Include FAQs when relevant
  - Demonstrate expertise and trustworthiness (E-E-A-T)
- Always include internal links to relevant pages
