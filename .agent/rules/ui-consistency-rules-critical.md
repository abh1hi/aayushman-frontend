### 4.1 Layout Consistency
- Use the same:
  - Grid system
  - Spacing scale
  - Section padding
  - Typography hierarchy
- Page layouts must follow predefined templates:
  - Hero section
  - Value proposition
  - Content sections
  - CTA section
  - Footer

Never invent new layout patterns without explicit instruction.

---

### 4.2 Typography Rules
- Use only approved fonts
- Maintain consistent:
  - Font sizes
  - Line heights
  - Font weights
- Headings and body text styles must be identical site-wide

---

### 4.3 Color & Branding Rules
- Use only approved brand colors
- Maintain consistent:
  - Button styles
  - Hover states
  - Link styles
  - Icon usage
- No random visual variations across pages
